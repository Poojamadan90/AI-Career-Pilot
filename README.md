# 🤖 AI Career Pilot

AI Career Pilot is an AI-powered Resume Analyzer that helps job seekers improve their resumes using Google Gemini AI.

It analyzes resumes, calculates an ATS score, identifies existing skills, highlights missing skills, suggests improvements, generates a personalized learning roadmap, and provides interview questions based on the user's target role.

---

## 🚀 Features

- User Login & Signup
- Resume Upload (PDF)
- Resume Text Analysis
- AI-powered ATS Score
- Skills Detection
- Missing Skills Analysis
- Resume Strengths
- Improvement Suggestions
- Career Roadmap
- Interview Questions
- Resume History

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Poojamadan90/AI-Career-Pilot.git
```

Move into the project

```bash
cd AI-Career-Pilot
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=your_api_key_here
```

Run the project

```bash
python app.py
```
